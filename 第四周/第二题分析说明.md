# 第二题分析说明

## 1. 变量选择思路

本题以 `Exam_Score` 作为因变量，选取了 9 个与学生个人学习特征较相关、且缺失值较少的自变量：

- `Hours_Studied`：学习时长
- `Attendance`：出勤率
- `Previous_Scores`：过往成绩
- `Tutoring_Sessions`：课外辅导次数
- `Sleep_Hours`：睡眠时长
- `Motivation_Level`：学习动机水平
- `Parental_Involvement`：家长参与程度
- `Internet_Access`：是否具备网络条件
- `Learning_Disabilities`：是否存在学习障碍

选择这几项的原因是：

- 它们都与学生个人学习状态、学习支持条件或学习基础直接相关。
- 这些变量既包含数值变量，也包含分类变量，能够较完整地描述学生成绩差异。
- 这组变量没有明显缺失问题，建模过程更稳定。

## 2. 数据处理与建模方法

建模前，对分类变量使用了哑元编码（Dummy Coding），把文字类别转换为可进入线性回归模型的 `0/1` 变量。

本次哑元变量的参考组为：

- `Motivation_Level` 的参考组：`High`
- `Parental_Involvement` 的参考组：`High`
- `Internet_Access` 的参考组：`No`
- `Learning_Disabilities` 的参考组：`No`

模型构建流程如下：

1. 从原始数据中提取上述 9 个自变量和 `Exam_Score`。
2. 对分类变量做哑元编码。
3. 将数据按 `80%` 训练集、`20%` 测试集划分。
4. 使用 `LinearRegression` 建立多元线性回归模型。
5. 使用 `R^2` 和 `RMSE` 评价模型拟合效果与泛化能力。
6. 使用 `VIF` 检查多重共线性。
7. 使用 `OLS` 输出系数显著性结果，辅助解释变量作用。

## 3. 回归方程

模型的近似表达式为：

```text
Exam_Score = 41.972
+ 0.292 * Hours_Studied
+ 0.199 * Attendance
+ 0.049 * Previous_Scores
+ 0.514 * Tutoring_Sessions
- 0.029 * Sleep_Hours
- 1.042 * Motivation_Level_Low
- 0.540 * Motivation_Level_Medium
- 1.956 * Parental_Involvement_Low
- 1.056 * Parental_Involvement_Medium
+ 0.899 * Internet_Access_Yes
- 0.881 * Learning_Disabilities_Yes
```

说明：

- 对于数值变量，系数表示在其他变量不变时，该变量每增加 1 个单位，`Exam_Score` 平均变化多少。
- 对于哑变量，系数表示该类别相对参考组的平均差异。

## 4. 模型结果

### 4.1 拟合与泛化能力

- 训练集 `R^2 = 0.6342`
- 测试集 `R^2 = 0.6886`
- 训练集 `RMSE = 2.3719`
- 测试集 `RMSE = 2.0979`
- 全样本 `OLS R^2 = 0.6446`
- 调整后 `R^2 = 0.6440`

这些结果说明：

- 模型大约可以解释 `64%` 左右的成绩波动。
- 测试集 `R^2` 与训练集 `R^2` 比较接近，说明模型在新样本上的表现较稳定，泛化能力较好。
- `RMSE` 约为 `2` 分左右，表示模型预测值与真实成绩之间的平均误差不大。

### 4.2 多重共线性检验

主要自变量的 `VIF` 值如下：

- `Hours_Studied`: `1.0020`
- `Attendance`: `1.0021`
- `Previous_Scores`: `1.0027`
- `Tutoring_Sessions`: `1.0011`
- `Sleep_Hours`: `1.0015`
- `Motivation_Level_Low`: `1.7468`
- `Motivation_Level_Medium`: `1.7479`
- `Parental_Involvement_Low`: `1.3588`
- `Parental_Involvement_Medium`: `1.3573`
- `Internet_Access_Yes`: `1.0013`
- `Learning_Disabilities_Yes`: `1.0016`

通常认为 `VIF < 5` 时，多重共线性问题不明显。本模型的各项 `VIF` 都较低，因此可以认为自变量之间不存在严重共线性。

## 5. 关键变量解释

从回归系数和显著性结果来看：

- `Hours_Studied` 系数为 `0.2916`，显著为正，说明学习时长越多，成绩通常越高。
- `Attendance` 系数为 `0.1994`，显著为正，说明出勤率越高，成绩越好。
- `Tutoring_Sessions` 系数为 `0.5136`，显著为正，说明适当辅导对提升成绩有积极作用。
- `Previous_Scores` 系数为 `0.0495`，显著为正，说明以往学习基础较好的学生，本次成绩通常也更高。
- `Internet_Access_Yes` 系数为 `0.8988`，显著为正，说明有网络条件的学生平均成绩更高。
- `Learning_Disabilities_Yes` 系数为 `-0.8810`，显著为负，说明存在学习障碍的学生在成绩上可能面临额外困难。
- `Motivation_Level_Low` 和 `Motivation_Level_Medium` 系数均为负，且相对于参考组 `High`，说明学习动机较低的学生成绩更低。
- `Parental_Involvement_Low` 和 `Parental_Involvement_Medium` 系数均为负，且相对于参考组 `High`，说明家长参与程度越低，成绩越低。
- `Sleep_Hours` 系数为 `-0.0294`，且 `p = 0.4889`，在本模型中不显著，说明其线性影响不明显。

## 6. 结果结论

综合来看，本次模型表明：

- 学习投入类因素，如学习时长、出勤率、辅导次数，对成绩提升最直接。
- 学习基础和学习支持条件，如过往成绩、网络条件，也对成绩有正向作用。
- 学习动机和家长参与程度较低，会显著拉低考试成绩。
- 学习障碍因素需要在教学中额外关注。

## 7. 可写入作业的简短结论

本研究使用学生学习行为与背景变量构建了多元线性回归模型，以 `Exam_Score` 为因变量进行预测。结果显示，模型在测试集上的 `R^2` 为 `0.6886`，说明模型具有较好的解释力与泛化能力；同时，各变量 `VIF` 均小于 `2`，表明不存在明显多重共线性问题。学习时长、出勤率、辅导次数、既往成绩和网络条件对考试成绩具有正向影响，而学习动机较低、家长参与程度较低及存在学习障碍则与较低成绩相关。总体上，该模型能够较好揭示学生成绩差异的主要影响因素，并为后续教学干预提供依据。
